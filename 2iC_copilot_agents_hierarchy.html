Agent hierarchy

Every broker gets their own dedicated team of AI agents structured across three tiers. The broker only ever speaks to the Primary Agent — everything else happens behind the scenes.

Tier 1 Single point of contact — the broker speaks only to this agent
Primary Agent
Understands the broker's full context across all active deals. Receives every message, decides which departments to route to, and returns a single coherent response.

Three-layer knowledge profile — read by every agent before every conversation
Layer 1 — 2iC base
Core industry knowledge, compliance obligations, communication standards. Maintained by 2iC. Applies to all brokers across all brands.
Layer 2 — Brokerage layer
NxtWave-specific config: preferred lenders, brand voice, internal processes. Set once on onboarding. Inherited by all NxtWave brokers.
Layer 3 — Broker profile
Individual to each broker. Built through onboarding conversation, refined through daily use. Personal preferences and working rules.
